# Тесты для лабораторной работы №05
Задача 2
## Входные данные
файл с вещественными числами.

## Выходные данные
выполняется ли правило трех сигм (1 - да; 0 - нет).

## Позитивные тесты
 - 01 - все числа разные;
 - 02 - все числа одинаковые;
 - 03 - несколько единиц и число 1 000 000.

## Негативные тесты
 - 01 - в файле буквы;
 - 02 - пустой файл.
